# The Untravel Show

Starter draft website for:

- Story-led travel episodes (YouTube Shorts, Instagram Reels, long-form YouTube)
- Curated fixed-date group trips (6-7 participants, selection-based)
- On-demand Airbnb Experiences
- Direct support options (one-time and recurring)

## Local preview

Open `index.html` directly in your browser.

## Deploy to Netlify

1. Push this repository to GitHub.
2. In Netlify, create a new site from Git and select this repo.
3. Build command: *(leave empty)*.
4. Publish directory: `.`.

`netlify.toml` is included for SPA-style fallback routing.
